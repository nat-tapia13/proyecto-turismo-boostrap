## ✨ Descripción

Sitio web de turismo moderno inspirado en plataformas profesionales, desarrollado con enfoque en **diseño responsive**, **interactividad** y **experiencia de usuario**.

## 📱 Diseño Responsive

- 📱 Mobile → 1 columna  
- 📲 Tablet → 2 columnas  
- 💻 Desktop → 3 columnas  


## ⚙️ Funcionalidades

- Filtros dinámicos con jQuery  
- Cards con efectos hover  
- Carrusel de testimonios  
- Sistema de rating  
- Comentarios simulados  
- Validación de formularios  

## 🎨 Tecnologías

- HTML5  
- CSS3  
- Bootstrap 5  
- JavaScript  
- jQuery  


## 🎯 Sprite CSS

- Uso de `background-image`  
- Uso de `background-position`
